##LC12. Integer to Roman

Given an integer, convert it to a roman numeral.
Input is guaranteed to be within the range from 1 to 3999.
罗马数字拼写规则：

罗马数字共7个：I（1）、V（5）、X（10）、L（50）、C（100）、D（500）和M（1000）。可以表示任意正整数。

@(hui英新1)
以下是规则：

  • 重复次数：一个罗马数字重复几次，就表示这个数的几倍。
  • 右加左减：
    • 左大右小，则大数加小数；
    • 左小右大，则大数减小数；
    • 左减的数字有限制，仅限于( I X C )。比如45不能写成VL，只能是XLV。
    • 左减不可以跨越一个位值。比如99不可以使用IC（100-1）表示，而是用XCIX（100-10）+ （10-1）表示。等同于阿拉伯数字每位数字分别表示。
    • 左减数字必须为1位，比如8写成VIII，而不是IIX。
    • 右加数字不可以连续超过3位，比如14写成XIV，而非XIIII。
  • 加线乘千：在罗马数字的上方加上一条横线或者加上下标的M表示将该数乘以1000.
  • 同理：如果上面有两条线，则是原数的1,000,000倍。
  • 数码限制：
    • 同一个数码最多只能连续出现3次，如40不能表示为XXXX（10+10+10+10），而要表示为XL（50-10）。
    • 例外：由于IV是古罗马神话主神朱庇特的首字，因此有使用IIII代替IV来表示4。
